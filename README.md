# Tripadvisor Clone (HTML & CSS)

## 📌 Project Overview
This is a basic clone of the Tripadvisor homepage built using only HTML and CSS.  
The goal of this project is to practice frontend development fundamentals without using JavaScript or frameworks.

## 🚀 Features
- Responsive Navigation Bar
- Hero Section with Search Bar
- Category Cards (Hotels, Things to Do, Restaurants, Flights)
- Popular Destinations Grid
- Clean and Modern UI
- Flexbox and CSS Grid Layout
- Sticky Navbar

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
